# Personal Expense Analysis

## Problem Statement
Understanding personal spending patterns is difficult without structured analysis.
This project analyzes transaction-level expense data to identify spending trends,
high-expense categories, and monthly expense behavior.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Monthly expense aggregation
- Category-wise expense analysis
- Average daily spending calculation
- Identification of highest spending month and category

## Key Insights
- Identified the month with maximum expenses
- Found the category contributing the most to overall spending
- Calculated average daily expense to understand burn rate

## Outcome
This analysis helps users gain clarity on where their money is spent and
supports better financial decision-making.
